Convergence of graph Laplacian to smooth Laplacian

2019/08/20

Let's write some \(\LaTeX\) first. \( \newcommand{\R}{\mathbb{R}} \)

I want to consider an embedded closed submanifold \(\Sigma\) in an ambient Euclidean space \(\R^N\).

\(\Sigma\) inherits the ambient space's metric from which we can build such objects as the smooth Laplacian \(\Delta\). I shall pick \(n\) points \(\{x_i\}\) randomly from \(\Sigma\). This provides a graph \(G\) from which I will build an \(n\times n\) matrix called the graph Laplacian.

Any smooth function \(f\) on \(\Sigma\) can also be seen as function on \(G\) by evaluation at each point \(x_i\).

Hopefully we can rewrite the maths of this paper to show that as \(n\to\infty\), the two Laplacians converge to eachother pointwise.
